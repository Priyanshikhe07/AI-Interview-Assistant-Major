# 🤖 AI Interview Assistant

An AI-powered Resume Screening & Interview Preparation Platform built using **Python**, **Streamlit**, and **Google Gemini AI**.

🔗 **Live Demo:**  
https://ai-interview-assistant-priyansh.streamlit.app

---

## 🚀 Features

- 📄 Upload Single or Multiple Resume PDFs
- 📊 ATS Score Analysis
- 🎯 Resume vs Job Description Matching
- 🤖 AI Generated Technical Interview Questions
- 👥 Batch Resume Screening for Recruiters
- 🛠 Skill Extraction
- 📈 Match Score Calculation
- ⚡ Fast Resume Parsing
- 🎨 Modern Streamlit UI

## 💡 How It Works

1. Upload one or multiple resumes.
2. Paste the Job Description.
3. Analyze ATS Score.
4. Compare Resume with JD.
5. Generate personalized AI Interview Questions.
6. Recruiters can evaluate multiple candidates simultaneously.

# AI Interview Assistant

An AI-powered interview preparation tool built with Streamlit and Google Gemini API.

## Features
- Upload Resume (PDF)
- Extract Resume Text
- Generate AI-based Technical Interview Questions
- Interactive Streamlit UI

## Tech Stack
- Python
- Streamlit
- Google Gemini API
- PyPDF2

## Run Locally

pip install -r requirements.txt
streamlit run app.py
